How to Use New Features (Short Practical Notes)

Settings page:

Go to Tools → URE Settings.

Adjust batch sizes, preview limits, history limits, and backup retention.

Toggle features (logging, AJAX, warnings).

Save settings.

Saved profiles:

Configure your search/replace job on the main page.

Scroll below the Operation History to the Saved Profiles section.

Save current settings with a profile name.

Later: select the profile, click Load, and the form auto-fills.

Help:

Go to Tools → URE Help.

Use it as a built-in manual and troubleshooting guide.

Advanced Database Mode:

On the main page, scroll below Saved Profiles.

Click “Open Advanced Database Mode” to reveal the database tools.

Create backups first, then run database-level operations.

Click “Close Advanced Database Mode” to hide the panel.
